## 港股数据源

### 1. HKEX 官方行情

#### HKEX OTC (Open Trading System) API

| 特性 | 说明 |
|------|------|
| **官网** | https://www.hkex.com.hk/ |
| **数据类型** | 现货、期货、期权、涡轮、牛熊证 |
| **延迟** | 15分钟延迟（免费） |
| **实时** | 需要付费订阅 |
| **协议** | HTTP REST / WebSocket |
| **数据格式** | JSON / XML |

**优点：**
- 官方权威数据源
- 覆盖所有港交所产品
- 可靠性高

**缺点：**
- 实时数据需付费（价格昂贵）
- 15分钟延迟行情免费但实用价值有限
- API文档为英文，门槛较高

#### HKEX Market Data Feed (MDF)

- 低延迟二进制协议
- 主要面向机构用户
- 需要专线连接

### 3. 免费/开源数据源

#### Tushare

| 特性 | 说明 |
|------|------|
| **官网** | https://tushare.pro/ |
| **数据** | A股、港股、美股、基金、期货 |
| **协议** | HTTP REST |
| **数据格式** | JSON / Pandas DataFrame |
| **认证** | Token |
| **免费额度** | 基础数据免费 |
| **Python支持** | 优秀 |

**优点：**
- 数据全面
- 文档完善
- Python接口友好
- 社区活跃

**缺点：**
- 实时性较差（批量获取）
- 高级数据需付费
- 请求频率有限制

**主要API：**

```python
import tushare as ts

# 设置token
ts.set_token('your_token')
pro = ts.pro_api()

# 查询日线数据
df = pro.daily(ts_code='000001.SZ', start_date='20240101', end_date='20240110')

# 查询实时行情
df = pro.realtime_quote(ts_code='000001.SZ')
```

```python
import akshare as ak

# 实时行情

df = ak.stock_zh_a_hist(symbol="000001", period="daily", start_date="20240101", end_date="20240110")

# 分时数据
df = ak.stock_zh_a_minute(symbol="000001", period='1', adjust="qfq")
```

## API特性对比

### 认证方式对比

| 数据源 | 认证方式 | 复杂度 | 安全性 |
|--------|----------|--------|--------|
| HKEX | Token/API Key | 中 | 高 |
| 富途 | Token | 低 | 高 |
| 华泰国际 | 数字证书 | 高 | 高 |
| Tushare | Token | 低 | 中 |
| AKShare | 无 | 极低 | 低 |
| 东方财富Choice | 账号密码 | 中 | 高 |

### 请求频率限制

| 数据源 | 频率限制 | 说明 |
|--------|----------|------|
| Tushare Pro | 2000次/分钟 | 基础用户 |
| AKShare | 无明确限制 | 但依赖数据源 |
| 富途 | 100次/秒 | WebSocket更高 |
| HKEX | 根据订阅级别 | 不等 |
| 东方财富 | 未知 | 可能限流 |

### 数据格式支持

| 数据源 | JSON | Protobuf | Pandas | 备注 |
|--------|------|----------|--------|------|
| 富途 | 是 | 是 | 是 | 多语言SDK |
| Tushare | 是 | 否 | 是 | Python优先 |
| AKShare | 否 | 否 | 是 | Python专用 |
| HKEX | 是 | 可选 | 需转换 | REST支持JSON |
| 阿里云 | 是 | 否 | 需转换 | REST |

### Go SDK支持情况

| 数据源 | 官方Go SDK | 第三方库 | 备注 |
|--------|------------|----------|------|
| 富途 | 有 | - | 完整支持 |
| Tushare | 无 | 第三方 | 功能有限 |
| AKShare | 无 | 无 | 建议Python |
| HKEX REST | 无 | 第三方 | 需自己封装 |
| 华泰国际 | 有 | - | Java优先 |

**Go量化相关库：**
- `github.com/lightweighttrader/gotushare` - Tushare Go客户端
- `github.com/toyzz/gostock` - 港股数据

## 推荐方案

### 方案一：低成本起步（个人投资者）

**数据源：**
- A股：AKShare（免费）
- 港股：Yahoo Finance（延迟）+ 富途（实时，需开户）

**工具栈：**
- Python + Pandas
- AKShare获取数据
- 富途OpenAPI获取港股实时

**优点：**
- 零成本
- 数据源丰富
- 开发简单

**缺点：**
- 数据稳定性一般
- 实时性有限

### 方案二：专业量化（进阶）

**数据源：**
- A股：Tushare Pro（付费）
- 港股：富途（实时）

**工具栈：**
- Python / Go
- Tushare Pro API
- 富途OpenAPI

**优点：**
- 数据质量好
- 支持实时
- SDK完善

**缺点：**
- 需要付费（tushare订阅）
- 需要香港账户

### 方案三：机构级（全市场）

**数据源：**
- A股：Wind万得 / 东方财富Choice
- 港股：HKEX直连 / 券商专线
- Level-2：交易所直连

**工具栈：**
- C++/Java/Go
- 自建数据 pipeline
- 专线/托管

**优点：**
- 数据最全面
- 延迟最低
- 可靠性最高

**缺点：**
- 成本极高
- 资质要求严格

## 总结

| 场景 | 推荐数据源 | 预算 | 难度 |
|------|-----------|------|------|
| 学习/研究 | AKShare + Yahoo | 0 | 低 |
| 个人量化 | Tushare Pro + 富途 | 中 | 中 |
| 机构交易 | Wind + HKEX直连 | 高 | 高 |
| 实时监控 | 富途 + AKShare混合 | 低 | 中 |

**关键决策点：**
1. 是否需要香港账户（影响港股数据源选择）
2. 是否需要实时数据（影响成本）
3. 是否需要Level-2（影响数据源）
4. 编程语言偏好（影响SDK选择）
